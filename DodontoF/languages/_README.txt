
言語ファイルの作り方

言語ファイルはこのディレクトリに

English.txt

のように「言語名.txt」のテキストを作成して作ります。
ただし このREADMEファイルのように「_」で始まるファイルは対象外となります。

言語名はここでは必ず英語にしておいてください。文字コード問題が出ますので。
画面に表示する言語名はTEXTファイル内部に記載します

テキストファイルの中身は

language = English
title = DodontoF
loginWindowTitle = Login

こんなフォーマットになっています。
左側は何の文字を差し替えるかのキーワードです。
右側が表示させる文字列。
たとえば最初のlanguage は言語の表示名を書きます。
titleはブラウザに表示されるタイトルですね。

「#」 で始まる行はコメント行のため無視されます。

ダイスボットの名前を変更する場合には

DiceBotName_ArsMagica = アルス☆マギカ

のように

DiceBotName_(ダイスボット種別名）= 変更したい名前

という定義を追加してください。
ダイスボット種別名は画面に表示される名前ではなく、
src_ruby/diceBotInfos.rb
で
gameType
として定義されているものです。そちらのファイルを参照してください。


もし多言語化ファイルが作成できたら是非どどんとふ公式に送ってあげてください。
